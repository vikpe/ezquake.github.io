Installing ezQuake

Follow these steps to get started with ezQuake.

1. Buy Quake on Steam
Quake is required in order to run ezQuake. This will give you the complete single-player game.

2. Download ezQuake (full install)
Extract the content to your Quake installation directory.

Alternative installation: nQuake

nQuake is a package that includes ezQuake, config files, modern HUD, fullbright skins and 24-bit textures to quickly get configured for competitive online games.
