Welcome to the home of ezQuake, a modern QuakeWorld client focused on competitive online play.

Combining the features of modern QuakeWorld® clients, ezQuake makes QuakeWorld® easier to start and play. The immortal first person shooter Quake® in a brand new skin with superb graphics and extremely fast gameplay.

Installing ezQuake

Full install include the bare minimum of game media (basic hud graphics, player skins etc). If you want to experience modern graphics and other additional media in the game including custom configurations, try nQuake.

{% include download-latest.html %}

Alternatively, you can build the client from source.

Installing Quake

  1. Buy Quake on Steam, this will give you the complete single-player game.
  2. Download a 'Full' install above and copy to your Quake directory.

TIP To get configured for competitive online games quickly, try nQuake. nQuake includes standard config files, modern HUD, fullbright skins and 24-bit textures. Once nQuake is installed, download an Upgrade version of ezQuake and replace the executable in your Quake directory.
